# Flappy Bird Clone 🐦

**Flappy Bird Clone** is a simple arcade game built with **Python** and **Pygame**. It features smooth gravity physics, procedural pipe generation, a scoring system, and instant restart logic.

### 🚀 Features
* **Gravity Physics:** Realistic falling and jumping mechanics.
* **Procedural Generation:** Pipes are generated dynamically as you play.
* **Clean Code:** An excellent example of game loops and collision detection.

### 🎮 How to Play
1. Install Pygame: `pip install pygame`
2. Run the game: `python main.py`

**Controls:**
* **W** or **Space** — Jump
* **R** — Restart the game

### 🛠 Tech Stack
* Python 3.x
* Pygame library
